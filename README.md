
# Todo App

Todo App using HTML, CSS, Javascript, NodeJs, and MongoDB.

Link to production site: [Todo App](https://powerful-overshirt-tick.cyclic.app/)


## Features

- Add Task by category and date
- Filter by category to list tasks
- Delete task
- Responsive App
